# Design Context — Rodak (tienda nueva)

## Design Context

### Users
Compradores argentinos de muebles premium de home-office y accesorios de setup (escritorios, elevaciones de monitor, estanterías, soportes, bandejas). Navegan en desktop y mobile, evaluando compras de ticket alto ($500k–$1.5M ARS). Quieren **visualizar su setup armado** y sentir confianza antes de una compra grande. Cultura "setup/productividad".

### Brand Personality
Premium, cálido-industrial, obsesionado con el setup. Materiales reales: madera maciza + hierro. Voz argentina informal (vos): "Diseñá tu setup". Aspiracional pero con los pies en la tierra. Tres palabras: **crafted, warm, precise**.

### Aesthetic Direction
**El mix explícito:** tomar el LAYOUT y los patrones de UX de balolo.de (editorial-minimal premium: mucho whitespace, fotografía de producto grande, hero cinematográfico, UI contenida, jerarquía tipográfica clara, sistema modular de producto) + la PIEL de Rodak (acento dorado/ámbar, wordmark Rodak, tinta carbón cálida, tonos de madera reales del catálogo).

- **Base:** off-white cálido tintado hacia el ámbar (no blanco puro).
- **Tinta:** carbón cálido casi negro (la identidad dark de Rodak se vuelve el texto, no el fondo).
- **Acento:** dorado/ámbar de Rodak, usado con moderación (regla 60-30-10).
- **Referencia:** balolo.de (estructura). **Anti-referencia:** WordPress/Woodmart genérico actual; AI-slop (cyan-on-dark, gradientes morados, cards con drop shadow genérico).
- **Tema:** claro (como balolo). Editorial, no dashboard.

### Design Principles
1. **La foto manda.** El producto (madera + hierro) es la estrella; el UI se corre para dejarlo respirar.
2. **El sistema modular es el héroe.** Comunicar "escritorio + accesorios que combinan" como un sistema, no como productos sueltos — es el diferencial vs. la tienda actual.
3. **Lujo por restricción.** Whitespace generoso, pocos tamaños tipográficos con mucho contraste, acento dorado escaso. Nada de decoración de relleno.
4. **Confianza para ticket alto.** Prueba social (reviews), señales (hecho en Argentina, envíos, cuotas reales), precios claros con descuento por transferencia.
5. **Argentino, no traducido.** Copy en castellano rioplatense, precios en ARS, medios de pago locales.

### Tipografía (fuera del molde)
- Display: **Bricolage Grotesque** (contemporáneo, con carácter, editorial-premium).
- Body: **Hanken Grotesk** (limpio, cálido, legible).
- (Evitar los defaults de siempre: Inter, DM Sans, Fraunces, Space Grotesk, etc.)
